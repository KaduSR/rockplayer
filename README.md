# RockPlayer 🎸

> Player de audio tematico rock com visualizacoes em tempo real e letras sincronizadas.

## Stack
React • Web Audio API • YouTube Data API • Canvas

## Funcionalidades
- Visualizacao em tempo real (Canvas 60fps)
- Busca no YouTube Data API
- Letras sincronizadas com a musica
- Playlist colaborativa
- Equalizador visual

## Metrica
Renderizacao de audio visualizado a 60fps no Canvas.
